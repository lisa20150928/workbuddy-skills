# 涨停上穿 60MA 选股技能

## 快速开始

筛选当日涨停且股价上穿 60 日均线的股票：

```bash
python scripts/filter_limit_60ma.py --date 2026-03-31 --output json
```

## 功能说明

本技能用于筛选同时满足以下两个条件的股票：

1. **当日涨停** - 股票在交易日达到涨停板
2. **上穿 60MA** - 当日收盘价上穿 60 日均线（技术突破信号）

### 上穿 60MA 的判断逻辑

- 当日收盘价 > 当日 60 日均线
- 前一日收盘价 ≤ 前一日 60 日均线

这是经典的技术分析突破信号，表明股价从均线下方突破至上方，可能开启上升趋势。

## 使用方法

### 基本用法

筛选最近交易日的涨停上穿 60MA 股票：

```bash
python scripts/filter_limit_60ma.py
```

### 指定日期

```bash
python scripts/filter_limit_60ma.py --date 2026-03-31
```

### 输出格式

**JSON 格式（默认）：**
```bash
python scripts/filter_limit_60ma.py --output json
```

**CSV 格式：**
```bash
python scripts/filter_limit_60ma.py --output csv
```

### 保存到文件

```bash
python scripts/filter_limit_60ma.py --date 2026-03-31 --output json --output-file result.json
```

## 数据来源

- 涨停数据：AKShare `stock_zt_pool_em` (东方财富涨停池)
- 历史行情：AKShare `stock_zh_a_hist` (A 股历史行情)

## 注意事项

1. **数据延迟** - 涨停池数据通常在交易日 16:00 后更新完整
2. **停牌股票** - 停牌股票无法获取历史行情，会自动跳过
3. **新股** - 上市不足 65 天的股票无法计算 60MA，会跳过
4. **网络请求** - 需要联网获取实时数据

## 典型使用场景

- **短线选股** - 识别涨停突破均线的强势股
- **打板策略** - 筛选有技术面支撑的涨停股
- **复盘分析** - 分析历史某日的涨停突破情况
- **趋势跟踪** - 寻找均线突破的启动点
